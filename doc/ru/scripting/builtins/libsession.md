# Библиотека *session*

Библиотека используется для хранения данных, не сбрасывающихся при загрузке/выгрузке контента, являясь рекомендованной альтернативой использованию таблицы `_G`.

Хранимые данные не сохраняются при завершении работы движка.

Для минимизации риска коллизий имён рекомендуется использовать префиксы паков (`id_пака:имя_таблицы`), либо, если пак хранит одну таблицу в *session*, использовать имя пака как имя записи.

```lua
-- Возвращает таблицу по имени. Создаёт пустую таблицу при отсутствии.
session.get(name: str) -> table

-- Удаляет таблицу по имени.
session.reset(name: str)

-- Возвращает true если таблица уже была создана вызовом session.get
session.has(name: str) -> boolean
```
